# Proposal: Make Report Title Dynamic

## Why
Currently, the PDF reports generated by the dashboard have a hardcoded title "Alfabetización Tecnológica" (or use the acronym "AFT" in footers). This prevents the system from being used for different types of assessments without code modifications.

## What Changes
1.  **Configuration**: New environment variables `PDF_REPORT_TITLE` and `PDF_REPORT_ACRONYM` are introduced.
2.  **PDF Generator**: The generation logic is updated to use these settings to draw the dynamic title in the template.

## Approach
1.  **Configuration**: Add a new environment variable `PDF_REPORT_TITLE` to the `.env` file and configure it in `project/settings.py`.
2.  **PDF Generation**: Update `utils/pdf_generator.py` to read the title from settings instead of using hardcoded values.
3.  **Dynamic Drawing**: Added logic to draw the dynamic title in its place on the `template.pdf`.

## Capabilities
- **dynamic-title**: The PDF report title and footer will reflect the value set in the environment variables.
